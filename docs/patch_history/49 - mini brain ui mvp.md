# 49 - Mini Brain UI MVP

Added a tiny browser-facing console for explaining and inspecting Mozok without needing to read raw Swagger schemas first.

## Added

- `GET /ui`
- `GET /ui/scenario-studio`
- `GET /ui/knowledge-graph`
- Inline HTML pages with links back to Swagger endpoints.

## Scope

This is intentionally a lightweight operator/demo UI, not a full frontend. It helps present the project to non-technical users and gives future UI work a stable starting point.
