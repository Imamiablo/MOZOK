# 26.3 - Final Pipeline Reranking Debug Hotfix

## Purpose

Patch 26 added the deterministic memory reranker and Patch 26.1 added the main debug report method. However, the final `final_prompt` pipeline step still did not expose the same `memory_reranking` report.

This hotfix adds the final reranking report to the final prompt pipeline step so `/debug/context` can show why final memories remained in the prompt.

## Changes

- Added `memory_reranking` to the `final_prompt` step in `ContextPackage.pipeline_steps()`.
- Added a regression test for the final pipeline step.

## Safety

This patch is debug-only. It does not change memory selection, memory storage, SQL records, FAISS, embeddings, or prompt content.
