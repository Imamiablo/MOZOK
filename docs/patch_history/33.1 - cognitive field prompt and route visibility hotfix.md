# 33.1 - Cognitive Field prompt and route visibility hotfix

## Summary

Small hotfix for the Cognitive Field MVP.

## Fixed

- The Cognitive Field report was present in debug output but its broadcast section was not rendered into the final system prompt.
- Added an explicit prompt section headed `Cognitive Field / broadcast focus for this turn` when the feature is enabled and a broadcast exists.
- Strengthened the OpenAPI test so it checks the served `/openapi.json`, which is what Swagger UI actually displays.

## Behaviour

The Cognitive Field remains read-only. The prompt section is soft attention guidance only; it does not authorise automatic memory, goal, entity-state, skill, relation, or FAISS changes.
