# 26.1 - Context Reranking Debug Hotfix

## Purpose

Patch 26 added the deterministic memory reranker, but in some working trees the
`ContextPackage` debug helper was not updated. This caused
`test_context_reranking_debug.py` to fail with:

```text
AttributeError: 'ContextPackage' object has no attribute 'memory_reranking_report'
```

## Changes

- Adds `ContextPackage.memory_reranking_report()`.
- Adds top-level `memory_reranking` to `ContextPackage.to_debug_dict()`.
- Adds reranking explanations to the retrieved and final pipeline steps.
- Does not modify SQL, FAISS, memory records, or prompt content.

## Tests

- `tests/unit/test_context_reranking_debug_hotfix.py`
- Existing Patch 26 tests should pass after this hotfix.

## Notes

This is a documentation and debug-output hotfix. It does not change reranking
scores or retrieval behaviour.
