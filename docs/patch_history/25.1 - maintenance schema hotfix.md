# 25.1 - Maintenance Apply Schema Hotfix

## Purpose

Patch 25 added maintenance apply/reject endpoints and the relation-aware protection service. In one branch, `mozok/schemas/memory.py` did not receive the new request/response models, so test collection failed with:

```text
ImportError: cannot import name 'MemoryMaintenanceApplyRejectRequest' from 'mozok.schemas.memory'
```

## Change

This hotfix adds the missing Pydantic schema models used by `mozok.memory.maintenance_apply` and the new API routes:

- `MemoryMaintenanceSuggestionInput`
- `MemoryMaintenanceApplyRejectRequest`
- `MemoryMaintenanceApplyRejectResult`
- `MemoryMaintenanceApplyRejectResponse`

The hotfix is applied by a small script so the existing `memory.py` file is not blindly overwritten.

## Files

- `scripts/apply_patch25_1_maintenance_schema_hotfix.py`
- `tests/unit/test_memory_maintenance_apply_schema.py`
- `mozok/docs/patch_history/25.1 - maintenance apply schema hotfix.md`

## Test

.\.venv\Scripts\python.exe -m pytest tests\unit\test_memory_maintenance_apply.py
